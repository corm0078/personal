Workland

App Design, UX

Workland is an app, I visioned after years of working retail jobs. I was frustrated with the scheduling aspect of working retail, with a paper schedule on the wall and no real way to request days off or change shifts with someone. I, therefore, envisioned an app which would make it easy to see your schedule request days and which days with a co-worker.

I Created an app interface, which would replicate a calendar app user experience but would allow the user to request days off, which shifts and get reminders for their upcoming shift. This interface was created with user workflow diagrams and wireframes. User testing was also executed on wireframes to create the best user experience. This app would simplify scheduling for managers and employees.
